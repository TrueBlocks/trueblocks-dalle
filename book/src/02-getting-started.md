# Getting Started

## Requirements
- Go 1.20 or newer
- macOS, Linux, or Windows

## Installation
Add DalleDress to your project with:

```sh
go get github.com/trueblocks/dalledress
```

No additional setup is required. All dependencies are managed via Go modules.

## Importing

```go

```
